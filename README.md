## About
Apply deep leaning algorithms over MNIST handwritten digits database, using Python 3.

## Requirements
- python (3.7.4)
- numpy (1.17.2)
- scikit-learn (0.21.3)
- scipy (1.3.1)
- matplotlib (3.1.1)
